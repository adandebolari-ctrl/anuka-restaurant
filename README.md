# Anuka Restaurant — Site web

Ce dossier contient un site web simple pour un restaurant, prêt à être utilisé comme base pour ZeroNet ou un autre hébergement statique.

## Fichiers

- `index.html` — page d'accueil
- `menu.html` — page du menu avec plats et prix en CFA
- `gallery.html` — page galerie photographique
- `about.html` — page de présentation du restaurant
- `contact.html` — page de contact et formulaire de réservation
- `style.css` — styles du site
- `script.js` — chargement des actualités et formulaire de réservation
- `data/posts.json` — actualités / événements

## Instructions

1. Si tu utilises ZeroNet, ouvre ton site dans l'application.
2. Ajoute un dossier `images/` et téléverse tes photos là-dedans.
3. Utilise les noms recommandés :
   - `images/hero.jpg`
   - `images/dish-mafe.jpg`
   - `images/dish-yassa.jpg`
   - `images/dish-dessert.jpg`
   - `images/interior.jpg`
4. Modifie l'adresse et le téléphone dans `about.html` et `contact.html`.
5. Publie le site en cliquant sur `Save`, `Sign` puis `Publish`.

## Personnalisation

- Change les plats et les prix dans `menu.html`.
- Ajoute ou remplace des actualités dans `data/posts.json`.
- Met à jour les images en téléversant tes propres photos.
